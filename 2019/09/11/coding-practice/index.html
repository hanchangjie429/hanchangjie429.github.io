Question21: Save the Prisoner

1. Difficulty: Easy
2. Solution

思考🤔：2019.9.17 今天重新想了想这道题，自己一开始的想法有点问题，因为是要找到最后一个拿到sweet的人，而且这个数列的元素是从1开始的，而不是从0开始的，所以取模的时候，余数为1代表是start_num（数列第1个元素）,余数为2代表是start_num右移1单位（数列第2个元素），……余数为n-1代表start_num右移n-2单位（数列倒数第2个元素），余数为0代表右移n-1单位（即数列最后1位），因此在算了m%n取模之后得到循环轮次后的偏移量（m=m%n)，根据这个偏移量再计算从start_num开始往后的偏移量，再进行取模操作得到偏移位上的数值(res=(m+s-1)%n）。因为如果余数为0说明是数列最后一位，即n. 但是这个思路没有下面的代码容易理解。

In this problem, we need to determine the ID of last prisoner which can be done by moving M-1 steps further from S.
ID of last prisoner= (M-1+S)
Since we are moving in a circle so we need to take mod of this summation with N.
Because the ID starts from 1, so the ID of last prisoner= (M - 1 + S - 1) % N + 1
