Summary

The algorithms that we are going to talk about today are Bubble sort, Insertion sort and Selection sort.

1.Bubble Sort(冒泡排序)

1. 排序原理

  • 冒泡排序只会操作相邻的两个数据
  • 对相邻两个数据进行比较，看是否满足大小关系，若不满足则将他俩互换位置
  • 一次冒泡排序会让至少一个元素移动到他应该在的位置，重复n-1次就能完成所有数据的排序工作，这也是为什么外循环次数是n-1次
  • 优化操作：若某次冒泡排序操作不存在数据交换，则说明数列已经达到完全有序状态

2. 代码实现（Python）

3. 性能分析

  • 执行效率：最小时间复杂度、最大时间复杂度、平均时间复杂度
    最小时间复杂度：数据完全有序时，只需一次冒泡操作即可，时间复杂度是O(n)
    最大时间复杂度：数据完全倒序时，需要n-1次冒泡操作，时间复杂度是O(n^2)
    平均时间复杂度：O(n^2) 待讨论

  • 空间复杂度：每次交换仅需一个临时变量，不申请额外的内存空间，故空间复杂度为O(1)，是原地排序算法

  • 算法稳定性：如果两个值相等，就不会交换位置，故是稳定排序算法

2.Insertion Sort(插入排序)

1.排序原理

  • 将数组中的数据分为两个区间，已排序区间和未排序区间
  • 初始已排序区间只有一个元素，即数组的第一个元素
  • 依此取未排序区间的元素，在已排序区间找合适的位置插入，保证已排序区间的元素一直有序
  • 重复操作直至未排序元素为空

2. 代码实现(Python)

3. 性能分析

  • 时间复杂度：最小时间复杂度、最大时间复杂度、平均时间复杂度

  • 空间复杂度：和冒泡排序一样，不需要任何额外的内存空间，因此空间复杂的为O(1),是原地排序算法

  • 算法稳定性：在插入排序中，对于值相同的元素，我们可以选择将后面出现的元素，插入到前面元素的后面，这样就保持原有的顺序不变，所以是稳定的

def selection_sort(mylist):
    for i in range(0,len(mylist)):
        min_index=i
        min_value=mylist[i]
        for j in range(i,len(mylist)):
            if mylist[j]<min_value:
                min_value=mylist[j]
                min_index=j
        mylist[i],mylist[min_index]=mylist[min_index],mylist[i]

    return mylist

array = [5, 6, -1, 4, 2, 8, 10, 7, 6]
print(selection_sort(array))
[-1, 2, 4, 5, 6, 6, 7, 8, 10]

def insertion_sort(mylist):
    for i in range(1,len(mylist)):
        for j in range(i-1,-1,-1):
            if mylist[j]>mylist[j+1]:
                mylist[j],mylist[j+1]=mylist[j+1],mylist[j]
    return mylist

array = [5, 6, -1, 4, 2, 8, 10, 7, 6]
print(insertion_sort(array))
[-1, 2, 4, 5, 6, 6, 7, 8, 10]

三步搞定冒泡排序

1. 弄清楚外循环次数 i

外循环次数等于 数组长度-1

2. 弄清楚内循环次数 j

内循环次数等于 列表长度-1-外循环计数

3. 弄清楚换值条件 if

换值条件为前一个数大于（或小于，取决于你的冒泡排序是顺序还是逆序）后一个数

outer loop: i = 0 to n-1
inner loop: j = 0 to n-1-i
